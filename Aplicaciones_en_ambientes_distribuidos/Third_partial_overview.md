# Javascript

## Objects
|   Type   | Object | Primitive |
| :------: | :----: | :-------: |
| Booleans |   ✓    |     ✓     |
| Numbers  |   ✓    |     ✓     |
| Strings  |   ✓    |     ✓     |
|  Dates   |   ✓    |           |
|  Maths   |   ✓    |           |
|  Regex   |   ✓    |           |
|  Array   |   ✓    |           |
| Function |   ✓    |           |
| Objects  |   ✓    |           |

*A Javascript object is an unordered collection of variables called **named values***

```js
var person = { firstName: "John",
lastName: "Doe",
age: 50,
eyeColor: "blue" };
```

## Comparison operators

| Operator | Condition                |
| -------- | ------------------------ |
| ==       | equal to                 |
| ===      | equal value and type     |
| !=       | not equal                |
| !==      | not equal value and type |
